## 0. 全局视觉规范（现代风 + 毛玻璃）

### 0.1 设计风格定位

| 维度 | 定义 |
|------|------|
| 整体气质 | 2024–2026 现代医疗健康小程序；轻盈、通透、专业可信；避免厚重拟物与过时渐变 |
| 布局原则 | 大留白、卡片化信息分组、清晰层级；主操作区固定底部安全区 |
| 圆角体系 | 小元素 8px / 卡片与按钮 16px / 大面板与弹层 20–24px |
| 阴影 | 柔和弥散阴影，非硬边；毛玻璃卡片用「内高光 + 轻外阴影」营造浮层感 |

### 0.2 色彩 Token

| Token | 色值 | 用途 |
|-------|------|------|
| `--bg-base` | `#E8F4F8` → `#F0F7FA` 浅青灰渐变 | 页面底层背景 |
| `--bg-mesh` | 低饱和蓝绿 `#B8E0E8`、薄荷 `#D4EDE4` 模糊光斑 | 背景装饰层（毛玻璃后方） |
| `--glass-fill` | `rgba(255,255,255,0.55)` ~ `0.72` | 毛玻璃面板填充 |
| `--glass-stroke` | `rgba(255,255,255,0.85)` 1px | 玻璃边缘高光描边 |
| `--text-primary` | `#1A2B3C` | 主标题、关键数字 |
| `--text-secondary` | `#5C6B7A` | 说明、次要信息 |
| `--accent` | `#2AAB9B` 青绿 | 主按钮、进度、选中态 |
| `--accent-soft` | `rgba(42,171,155,0.15)` | 标签底、选中背景 |
| `--success` | `#34C759` | 已完成、打卡 |
| `--warning` | `#FF9F0A` | 部分完成、待办 |
| `--wechat` | `#07C160` | 仅微信登录按钮 |

### 0.3 毛玻璃（Glassmorphism）组件定义

**核心参数（AI 生图与前端实现共用参考）：**

```
backdrop-filter: blur(20px) saturate(180%);
-webkit-backdrop-filter: blur(20px) saturate(180%);
background: rgba(255, 255, 255, 0.65);
border: 1px solid rgba(255, 255, 255, 0.8);
box-shadow: 0 8px 32px rgba(26, 43, 60, 0.08),
            inset 0 1px 0 rgba(255, 255, 255, 0.9);
border-radius: 16px;
```

| 组件 | 毛玻璃等级 | 说明 |
|------|------------|------|
| **GlassCard** | 标准（blur 20px，opacity 0.65） | 列表项、计划卡片、统计块 |
| **GlassNav** | 强（blur 24px，opacity 0.75） | 顶部导航栏、TabBar 底栏 |
| **GlassSheet** | 中（blur 16px，opacity 0.72） | 底部 ActionSheet、评分弹层 |
| **GlassInput** | 轻（blur 12px，opacity 0.5） | 搜索框、表单输入底 |
| **GlassFAB** | 标准 + accent 描边 | 浮动「+」按钮，略透明白底 |
| **GlassVideoFrame** | 弱毛玻璃边框包裹 | 视频区外一圈玻璃描边，内为实色视频占位 |

**层次规则：** 背景渐变/光斑 → 毛玻璃卡片 → 实色主按钮（不透明，保证可点性）→ 最顶层弹层（更强 blur + 更高 opacity）。

**禁忌：** 不在视频画面、小字号正文上直接叠强 blur；文字区玻璃底 opacity 不低于 0.55，保证对比度 WCAG 友好。

### 0.4 字体与图标

- 标题：SF Pro / PingFang SC 风格，Semibold，18–22px  
- 正文：Regular，14–15px，`--text-secondary`  
- 数据强调：DIN 或等宽数字感，24–32px，`--accent`  
- 图标：线性圆角图标（2px 描边），与毛玻璃卡片同色系

### 0.5 动效气质（生图静态稿可暗示）

- 卡片微悬浮、按钮轻微光泽  
- Tab 切换、弹层上滑时背景模糊加深（示意即可）  
- 进度条使用半透明玻璃轨道 + 实色 `--accent` 填充
