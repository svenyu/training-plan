# 脊柱侧弯康复训练小程序 - Code Wiki

## 一、项目概述

### 1.1 项目简介

本项目是一个**脊柱侧弯康复训练微信小程序**，面向家长用户，帮助孩子进行科学的脊柱康复训练。

**核心业务流程**：
```
家长微信登录 → 维护动作库（MinIO 视频）→ 制定计划（天数/日期范围）→ 开始训练打分 → 一次性订阅提醒
```

### 1.2 技术栈

| 分类 | 技术 | 版本 |
|------|------|------|
| 后端框架 | NestJS | ^10.4.22 |
| 前端框架 | UniApp (Vue 3) | 3.0.0-alpha |
| 数据库 | MySQL | 8.0+ |
| 对象存储 | MinIO | 最新稳定版 |
| ORM | Prisma | ^6.19.0 |
| 认证 | JWT + 微信 OAuth | - |
| 构建工具 | Vite | 5.2.8 |

### 1.3 环境要求

| 软件 | 版本要求 |
|------|----------|
| Node.js | ≥ 18.18（推荐 20.x LTS） |
| npm | ≥ 9 |
| MySQL | 8.0+ |
| MinIO | 最新稳定版 |
